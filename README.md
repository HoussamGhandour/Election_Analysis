# Election_Analysis
## Overview of Election Audit

The Election Commision requested our team to conduct additional analysis to complete the election audit. In particular the commision asked for the following analyses:

  - Voter turnout by County name
  - Percentage of votes by County name
  - County with highest voter turnout

## Election Audit Results

The summary of the results is as shown below.

**Total Votes:** 369,711

**Votes by County:**  
 - Jefferson: 10.5% (38,855)  
 - Denver: 82.8% (306,055)  
 - Arapahoe: 6.7% (24,801)  

**Largest County Turnout:** Denver

## Election Audit Summary

The script developed can work for any upcoming election given the results would be summarized in a csv file and the headers are maintained in the same position. Note that this script should be modified with new file path and file name depending on the new data to be analyized in the future.
